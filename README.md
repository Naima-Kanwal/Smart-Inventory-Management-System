## Features
- User Login System
- Add Products
- Update Products
- Delete Products
- Search Products
- Inventory Reports
- Low Stock Alerts
- File Storage using CSV

## OOP Concepts Used
- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
- Friend Functions
- Operator Overloading
- Static Members
- Constructors and Destructors

## Technologies
- C++
- File Handling
- Object-Oriented Programming
