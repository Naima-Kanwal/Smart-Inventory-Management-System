# Smart Inventory Management System

A console-based Inventory Management System developed in C++ using Object-Oriented Programming principles.

## Features
- User Login System
- Add Products
- Update Products
- Delete Products
- Search Products
- Inventory Reports
- Low Stock Alerts
- File Storage using CSV

## OOP Concepts Used
- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
- Friend Functions
- Operator Overloading
- Static Members
- Constructors and Destructors

## Technologies
- C++
- File Handling
- Object-Oriented Programming

## How to Run

1. Clone the repository
2. Open the project in any C++ IDE
3. Compile and run `main.cpp`
4. Login and manage inventory records
